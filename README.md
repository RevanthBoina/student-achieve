# Student Achieve

Student Achieve (Student Book of World Records) — a place to submit, verify, and showcase student achievements.

## Quick start (local development)

Requirements: Node.js and npm (or pnpm/yarn)

```sh
# Clone the repository
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start development server
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- Tailwind CSS

## Deployment

Build the project with `npm run build` and deploy the contents of the `dist/` folder to your preferred static hosting provider (Netlify, Vercel, GitHub Pages, etc.).

If you use a custom domain, configure DNS and point it to your host according to the provider's docs.
